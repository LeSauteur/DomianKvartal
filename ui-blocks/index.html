Изолированные блоки для сайта недвижимости

📦 Состав модуля

  • css/blocks.css — все стили блоков
  • js/blocks.js — интерактивность (скроллеры, избранное)
  • blocks.html — HTML-разметка всех блоков
  • assets/images/ — изображения для блоков

🔌 Подключение

  • Обернуть блоки в <div class="ui-blocks-wrapper">
  • Подключить css/blocks.css
  • Подключить js/blocks.js (для drag-скролла и кнопок)
